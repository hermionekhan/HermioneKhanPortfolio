Dynamometer Audio Exhibit & Factfile (2022)

Project Report

Designed from a brief by the National Railway Museum in York, this audio exhibit aims to simulate the interior of a Dynamometer car in use. Scored a second (63%)

This UI design includes a simple fact file and audio exhibit, conceptually displayed on a double-sided standing screen. In theory, the user would walk up to the screen situated near the Dynamometer Car carriage in the National Railway Museum, learn some quick historical facts before wearing the pair of headphones provided to be transported to an audio scenario.

For a first-year project, I was pleased to experiment and build my skills in UI design and audio design using this project. If I had to re-design and improve on any of my University Projects, I would start with this one to include Narratology and increased immersive techniques.

Skills Used

  • User Interface Design, Interaction Design & Graphic Design using Adobe Xd
  • Audio Recording, Engineering & Implementation using Adobe Audition & Premier Pro
  • Product & User Research Methods
  • Designing for a Client
  • Hardware Design & Ideation
